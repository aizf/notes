高度顺序：

1. 正z-index
2. `z-index:auto`或`z-index:0`
3. inline/inline-block盒子
4. float
5. block盒子
6. 负z-index
7. 层叠上下文(background/border)

元素层叠水平相当时的规则：

- 后来居上原则
- 谁 z-index 大，谁在上的准则

## 注意

首先，`z-index`属性值并不是在任何元素上都有效果。它仅在定位元素（定义了position属性，且属性值为非static值的元素）上有效果。
