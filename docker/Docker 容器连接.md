## Docker容器连接

端口映射并不是唯一把 docker 连接到另一个容器的方法。

docker 有一个**连接系统**允许将**多个容器连接在一起**，共享连接信息。

docker 连接会创建一个**父子关系**，其中父容器可以看到子容器的信息。

### 容器命名

当我们创建一个容器的时候，docker 会自动对它进行命名。另外，我们也可以使用 --name 标识来命名容器
